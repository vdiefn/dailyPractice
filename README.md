# 每日任務練習

這個儲存庫專門用來記錄與存放每日練習題與相關程式碼。主要目的為培養每日持續撰寫程式的習慣，並加強 JavaScript 與 Node.js 的後端核心邏輯熟練度。

---

## 🛠️ 開發環境與技術棧

* **執行環境：** Node.js
* **主要語言：** JavaScript
* **練習範疇：** HTTP 伺服器建立、API 設計、Git 版本控制、非同步處理
